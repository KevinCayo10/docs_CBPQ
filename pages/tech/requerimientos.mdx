# Manual Técnico del Sistema

## Introducción

Este documento describe los aspectos técnicos de un aplicativo móvil, diseñado para proporcionar funcionalidades específicas como gestión de inspecciones. El sistema está compuesto por un backend desarrollado en Laravel 11 y un frontend móvil realizado con React Native.

---

## Requerimientos Técnicos

### Aplicativo Móvil

- **Framework**: React Native.
- **Compatibilidad**:

  - **Sistema Operativo**: Android.
    - **Versión mínima**: Android 5.0 (Lollipop).
    - **Impresión**: Compatible únicamente con Android 11 o superior.
  - **Espacio de Almacenamiento**: Requiere al menos 100 MB libres.
  - **Conexión a Internet**: Obligatoria para la mayoría de las funciones.

- **Dispositivos Compatibles**:
  - Smartphones y tabletas Android.

### Backend

- **Framework**: Laravel 11.
- **Lenguaje**: PHP 8.2.
- **Servidor de despliegue**: Hosting compartido (cPanel).
- **Base de Datos**: MySQL, gestionada en el hosting.

### Hosting

- **Panel de Control**: cPanel.
- **Seguridad**: Certificados HTTPS instalados para garantizar conexiones seguras entre frontend y backend.

## Consideraciones Adicionales

- **Seguridad**:

  - **Protocolo HTTPS**: Todas las solicitudes al backend están aseguradas.
  - **Autenticación**: Uso de tokens para la comunicación segura entre clientes y servidor.

- **Compatibilidad**:
  - Aunque el sistema de impresión está restringido a Android 11 o superior, otras funcionalidades (como la generación de reportes y la administración de usuarios) están habilitadas para todas las versiones de Android desde la 5.0.
