# **Gestión de fichas de inspección**

Este módulo le ofrece la posibilidad de visualizar los productos tecnológicos Portátiles extraidos de las tiendas en líneas. Además, el sistema permite realizar filtros y ordenamientos con el fín de disminuir la cantidad y variedad de opciones.

#### 1. Acceder al modulo inspección

Se selecciona la opción del menu (icono) y se despliega un menu lateral. Por consiguiente se selecciona "Inspección"
